API for League of Legends

Description

The plan is to create a public and free to use API that provides data based on Riot Official API without ever replacing it.

Some examples of data I plan to provide are:

  • Champion specific winrate, pickrate, KDA, etc...
  • Champion matchups
  • Most used spells with each champion

Usage

The API key provided by Riot will be used to gather data.

Authentication

Due to potential abuse, I may implement an authentication system which utilizes api keys to rate limit users.

Riot Access

I commit to implement a system so that Riot can access data about the usage.
